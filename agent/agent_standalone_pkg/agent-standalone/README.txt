Intuit Tank Agent Standalone
--------------------------------------
This contains example files for a standalone agent for Intuit Tank. 
To deploy, 
1. Put these files on the machine that will serve as a stand alone agent. 
2. Ensure that the start_wats script is executable.
3. Modify the script to specify the controller url and optionally the capacity.
4. Start the agent with the run.sh script. e.g. nohup ./run.sh &
